== Description ==

Iqamah Times is the first fully-featured application to bring you the iqamah timings of your local masjids with a unique and user-friendly experience. This plugin provides a widget  for displaying daily athan and iqamah timings from iqamah.org

== Installation ==

1. Upload `adzan-and-qamah-times-a-simple-reminder` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add new Widget in 'Widgets' Section.

== Frequently asked questions ==

= What if I didn't found the masjid I search? =

You can register your masjid at Iqamah.org. It's Free!
